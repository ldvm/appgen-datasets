# Registry of contracts of the Czech Republic - Contracts

This is a sample data set for the D3.js Chord Visualizer. The data comes from
[OpenData.cz](http://www.opendata.cz), specifically
[this dataset](http://linked.opendata.cz/dataset/gov-cz-smlouvy).

Note that there is no single data set (named graph) that would contain all triples required by this
visualization. The attached TTL file contains hand-picked triples from the OpenData.cz triple store.
The data are not transformed in any way which means that a dedicated analyzer is required to
convert the data for the D3.js Chord Visualizer.
